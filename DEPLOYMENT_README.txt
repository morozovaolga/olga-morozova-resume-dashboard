=== ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ ПОРТФОЛИО ОЛЬГИ МОРОЗОВОЙ ===

Дата создания: 13 сентября 2025
Версия: Production Ready

СОДЕРЖИМОЕ АРХИВА:
==================
portfolio-dist.zip содержит папку dist/ с полностью собранным приложением.

СТРУКТУРА:
----------
dist/
├── index.html          - Главная страница (поддержка всех языков)
├── favicon.ico         - Иконка сайта
├── favicon.svg         - Векторная иконка
├── favicon-32x32.png   - PNG иконка
└── assets/             - Собранные ресурсы
    ├── index-*.css     - Стили (192KB)
    ├── index-*.js      - Основной код приложения
    ├── portfolioGenerator-*.js - Генератор DOCX (358KB)
    ├── react-*.js      - React библиотека
    ├── motion-*.js     - Анимации
    └── другие ресурсы

ЯЗЫКОВАЯ ПОДДЕРЖКА:
==================
✅ Русский язык (ru) - основной
✅ Английский язык (en) 
✅ Французский язык (fr)

Все переводы встроены в JavaScript файлы и работают без дополнительной настройки.

ФУНКЦИОНАЛЬНОСТЬ:
================
✅ Адаптивный дизайн для всех устройств
✅ Переключение между 3 языками
✅ 6 разделов портфолио (30+ проектов)
✅ Генерация DOCX-портфолио на лету
✅ Анимации и интерактивность
✅ SEO-оптимизация

ТРЕБОВАНИЯ СЕРВЕРА:
==================
- Любой веб-сервер (Apache, Nginx, IIS)
- Поддержка статических файлов
- HTTPS рекомендуется
- Поддержка MIME-типов для .js, .css, .html

УСТАНОВКА:
==========
1. Распаковать portfolio-dist.zip
2. Загрузить содержимое папки dist/ в корень веб-сайта
3. Настроить веб-сервер на обслуживание статических файлов
4. Проверить работу по адресу вашего домена

ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ:
=========================
- Приложение работает полностью в браузере
- Не требует серверной части или баз данных
- Совместимо с современными браузерами
- Размер архива: ~280KB (сжатый)
- Размер распакованного: ~1.2MB

ПОДДЕРЖКА:
==========
При возникновении проблем проверьте:
1. Все файлы загружены на сервер
2. Правильные MIME-типы для JS/CSS файлов
3. CORS настройки (если необходимо)

Версия Vite: 5.x
React версия: 18.x
TypeScript: 5.x
